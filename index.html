Sobre mi

Soy un desarrollador de software egresado como técnico superior en informática del IUT Región los Andes en el año 2006 con experiencia en diversas plataformas, abarcando desde aplicaciones web hasta el desarrollo en consolas. Mi pasión por la tecnología me impulsa a crear soluciones innovadoras y eficientes que se adaptan a las necesidades de cada proyecto, me gusta disfrutar el fútbol, soy un consumidor de videos en youtube y más cuando se trata de una tecnología que me llame la atención.

Poseo conocimientos en:

  • Programación en Java
  • Desarrollo web con Angular
  • Creación de API RESTful con Spring
  • Programación en AS400, incluyendo RPG, CL, DDS, así como la gestión de bases de datos con SQL y Query.
  • Manejo de bases de datos como PostgreSQL, MySQL
  • Dominio de PHP, JavaScript, HTML, y CSS

Actualmente, laboro para una empresa que presta servicios a la banca privada, donde contribuyo a desarrollar soluciones tecnológicas que optimizan los procesos y mejoran la eficiencia operativa de nuestros clientes.

Estoy comprometido a ofrecer productos de alta calidad y a estar siempre al día con las últimas tecnologías.

Proyectos

Proyecto Vulnerability

En este proyecto se mejoró la parte visual del sistema, se actualizó el jquery a la última versión, se realizó mejoras en el login para que no viajara el password del page hacia el servlet, sino que através de una librería de javascript encripte el password y luego es recibida por el servlet y convertida en texto nuevamente para enviarla al as400. Igualmente se suprimió la versión log4j obsoleta debido que presentaba vulnerabilidades en el sistema.

Trade Input

Se desarrolló todo lo concerniente a un producto para el manejo de inversiones, con todos los reportes que corresponden diario e implementados igualmente para generarse durante el cierre diario.

Custodia de Fondos Líquidos

Se desarrolló el módulo de custodia de fondos líquidos. En este módulo se colaboró en toda la parte web y java del sistema. Este módulo corresponde al manejo de recaudar todas la cuentas que no tienen movimiento durante más de 10 año y son recibidas para ser resguardadas en caso de aparición del titular o algún beneficiario.
